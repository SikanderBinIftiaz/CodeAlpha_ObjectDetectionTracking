# AI Object Detection & Tracking System

## 📌 Project Overview

This project was developed as part of the **CodeAlpha Python Programming Internship**.

It performs real-time object detection and tracking using the YOLOv8 model and ByteTrack algorithm.

---

## 🚀 Features

- Real-time object detection
- Multi-object tracking
- Live webcam support
- Video file support
- FPS counter
- Confidence score
- Automatic object tracking IDs
- Screenshot capture
- Save output video

---

## 🛠 Technologies Used

- Python
- OpenCV
- Ultralytics YOLOv8
- ByteTrack
- NumPy

---

## ▶ Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```
